Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito/destaque usando a tag <b>(não semântica,apenas forma)

Nesta frase tenho um termo em negrito/destaque usando a tag <strong>(semântica, possui sentido)

Itálico / Ênfase

Nesta frase, temos um termo em itálico/ênfase usando a tag <i>

(não semântica)

Nesta frase, temos um termo em itálico/ênfase usando a tag <em>(semântica)

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>

E no outro parágrafo, temos outro texto marcado no final.

Testo grande e pequeno

Estamos criando um texto grande(esta tag está obsoleta) e um texto pequeno neste parágrafo.

Texto excluido/ deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Testo incluido/ inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o texto sublinhado com a tag <u>. (não semântica)

Texto sobrescrito

Para inserir "coisas" sobre outras. ex: x3+x4

Texto subescrito

Para inserir "coisas" sob outras. ex: H2O
